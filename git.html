نسخه اندروید

لایسنس: Apache license 2.0

مخزن کد در گیت‌هاب: https://github.com/motmaenbash/motmaenbash-android

نسخه اندروید مطمئن باش تحت مجوز Apache 2.0 منتشر شده است.
استفاده، تغییر یا توزیع کد منبع آزاد است، اما استفاده از نام یا لوگوی «مطمئن باش» (MotmaenBash / Motmaen Bash) به‌طور کامل ممنوع است.
بیلد و انتشار عمومی پروژه تحت همین نام «مطمئن باش» فقط توسط تیم اصلی پروژه مجاز است.
